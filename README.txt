1. 表格文件名不能修改(新创建的文件的文件名最后有"_new")
2. 表格文件要跟.py代码放在同一个路径下
3. 需要以下扩展库的支持：xlrd  openpyxl  requests  lxml
4. 代码最开始用"######"包起来的部分是用来调节保存文件的频率的，用户可以根据需要进行修改
5. 代码运行开始由于要读取一个18万行的表格，所以会等待大概20秒钟左右
6. 测试的时候可以不让程序跑完，运行到至少保存了一次数据的时候可以随时终止程序。
7. 全民健康、寻医问药、药品通三个网站的抓取结果保存在相应的拼音命名的文件夹中(如果不存在，程序会创建)，.html文件名最后的数字代表其对应到表格中的行数，例如:QMJK_3.html这个文件表示表格中第三列（名称为“盐酸洛美沙星胶囊”）的对应抓取结果。特别说明：由于表格中第二行开始是药品数据的起始行，因此抓取的文件名也是从“XXX_2.html”开始。
